Unary Operators:

Unary operators operate on a single operand.

<script> let num = 5; num++; // Unary Increment (Postfix) console.log(num); // Outputs 6 ++num; // Unary Increment (Prefix) console.log(num); // Outputs 7 num--; // Unary Decrement (Postfix) console.log(num); // Outputs 6 --num; // Unary Decrement (Prefix) console.log(num); // Outputs 5 </script>

Binary Operators:

Binary operators operate on two operands.

<script> let operand1 = 10; let operand2 = 5; let addition = operand1 + operand2; // Binary Addition console.log(addition); // Outputs 15 let subtraction = operand1 - operand2; // Binary Subtraction console.log(subtraction); // Outputs 5 let multiplication = operand1 * operand2; // Binary Multiplication console.log(multiplication); // Outputs 50 let division = operand1 / operand2; // Binary Division console.log(division); // Outputs 2 </script>

Ternary Operator:

The ternary operator is a shorthand for an 'if-else' statement and operates on three operands.

<script> let age = 18; let eligibility = (age >= 18) ? 'Eligible' : 'Not Eligible'; // Ternary Operator console.log(eligibility); </script>
