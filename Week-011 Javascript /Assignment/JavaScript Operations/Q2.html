Arithmetic Operators:

<script> let num1 = 10;
let num2 = 5;
let addition = num1 + num2; // Addition
let subtraction = num1 - num2; // Subtraction
let multiplication = num1 * num2; // Multiplication
let division = num1 / num2; // Division
let modulus = num1 % num2; // Modulus (remainder after division)
console.log(addition, subtraction, multiplication, division, modulus); </script>

Assignment Operators:

<script> let x = 10; x += 5; // Equivalent to: x = x + 5 console.log(x); // Outputs 15 x -= 3; // Equivalent to: x = x - 3 console.log(x); // Outputs 12 </script>

Comparison Operators:

<script> let a = 10; let b = 5; let isEqual = a === b; // Strict equality let isNotEqual = a !== b; // Strict inequality let isGreaterThan = a > b; // Greater than let isLessThan = a < b; // Less than console.log(isEqual, isNotEqual, isGreaterThan, isLessThan); </script>

Logical Operators:

<script> let x = true; let y = false; let andResult = x && y; // Logical AND let orResult = x || y; // Logical OR let notResult = !x; // Logical NOT console.log(andResult, orResult, notResult); </script>

Unary Operators:

<script> let num = 5; num++; // Increment by 1 console.log(num); // Outputs 6 num--; // Decrement by 1 console.log(num); // Outputs 5 </script>

Ternary Operator:

<script> let age = 18; let eligibility = (age >= 18) ? 'Eligible' : 'Not Eligible'; console.log(eligibility); </script>

Bitwise Operators:

<script> let num1 = 5; // Binary: 0101 let num2 = 3; // Binary: 0011 let bitwiseAnd = num1 & num2; // Bitwise AND let bitwiseOr = num1 | num2; // Bitwise OR let bitwiseXor = num1 ^ num2; // Bitwise XOR console.log(bitwiseAnd, bitwiseOr, bitwiseXor); </script>
